# 隐私政策

**最后更新日期：2026年5月**

欢迎您使用食鉴！我们非常重视您的隐私保护。本《隐私政策》旨在向您说明在您使用我们的服务时，我们如何收集、使用、存储和共享您的个人信息。

## 1. 我们如何收集您的个人信息
在您使用食鉴的过程中，为了向您提供核心服务，我们可能会收集您的以下信息：
* **账号信息**：当您注册或登录时，我们需要您提供**手机号码**或**邮箱地址**。
* **日志信息**：当您使用我们的服务时，系统可能自动记录您的操作日志、设备型号及 IP 地址，用于排查错误和保障账号安全。

## 2. 我们如何使用您的个人信息
我们收集的个人信息将仅用于以下场景：
* 验证您的身份，保障您的账号安全。
* 为您提供基础的服务和客服支持。
* 遵守相关法律法规及监管要求。

## 3. 信息共享与披露
未经您的明确授权，我们**不会**将您的个人信息出售或共享给任何第三方，除非：
* 法律法规或国家相关部门要求。
* 为维护食鉴及其用户的合法权益所必须。

## 4. 联系我们
如果您对本《隐私政策》有任何疑问或建议，请通过官方渠道联系我们的客服团队。
