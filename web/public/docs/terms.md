## 2. 服务说明

### 2.1 核心功能
食鉴是一款以"个人食鉴图"为核心、由 C 端用户共建的真实餐厅评价产品。核心功能包括：
- 通过高德 POI 确认真实店铺
- 完成包含菜品打分的有效实践
- 生成个人六档食鉴图（夯爆了/夯/顶级/人上人/NPC/拉完了）
- 公开评价沉淀为餐厅和菜品评价池
- 广场社交功能

我们保留随时修改、暂停或终止部分或全部服务的权利，无需事先通知您。服务变更不影响您在变更前已产生的合法权益。
