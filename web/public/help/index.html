食鉴 使用指南

食鉴是一款餐饮评价工具，帮助你记录、发现和分享真实的用餐体验。

1. 注册与登录

手机号 + 密码登录

  1. 打开食鉴，进入登录页
  2. 输入手机号（+86 格式）和密码
  3. 勾选同意《隐私政策》和《用户协议》
  4. 点击「登录」

首次使用该手机号登录时，系统会自动创建新账号。

短信验证码登录

  1. 在登录页点击「验证码登录」切换
  2. 输入手机号，点击「获取验证码」
  3. 输入收到的 6 位验证码
  4. 验证通过后自动登录

验证码 60 秒后可重新获取。

找回密码

  1. 在登录页点击「忘记密码？」
  2. 输入手机号，获取并验证验证码
  3. 设置新密码（8-32 位，不能包含手机号后 6 位）

2. 地图探索

地图页是食鉴的首页，点击底部 Tab「地图」进入。

搜索餐厅

  1. 点击地图顶部的搜索栏
  2. 输入店名、地址或区域关键词
  3. 从下拉结果中点击目标店铺
  4. 地图自动定位到该店铺位置

筛选餐厅

  1. 点击搜索栏下方的筛选按钮（城市 / 等级 / 分类）
  2. 选择筛选条件，点击「确定」
  3. 地图上仅显示符合条件的餐厅标记

查看餐厅信息

  • 点击单个标记：弹出底部信息卡，显示店名、分类、地址、等级、评价人数
  • 点击聚类圆圈：地图放大，标记分散显示
  • 点击信息卡：跳转到餐厅详情页

3. 食鉴图

食鉴图是查看所有已评价餐厅的总览页面，点击底部 Tab「食鉴图」进入。

六档分级

食鉴将餐厅分为六个等级（从高到低）：

档位名称说明

顶级top优秀推荐

切换视图

点击右上角图标可在「网格视图」和「列表视图」之间切换。

搜索已评价门店

点击右上角搜索图标，输入关键词搜索自己评价过的餐厅。

开始食鉴

点击页面底部「开始食鉴」按钮，进入写评价流程。

4. 写评价（三步定档）

Step 1：搜索确认店铺

  1. 点击「开始食鉴」或餐厅详情页的「写评价」
  2. 选择搜索城市（可切换「全国」）
  3. 输入店名或地址关键词搜索
  4. 从结果列表中点击选定店铺

搜索无结果？点击页面下方的「手动补充」，填写店铺信息后继续。

Step 2：定档 + 评价

拖拽定档：

  1. 页面上方「待拖动区域」显示当前店铺卡片
  2. 按住店卡并拖动到目标档位，松手即选中
  3. 店卡会自动吸附到最近的档位

写整店锐评（选填）：

  • 在档位下方的输入框中写一句话评价
  • 点击骰子图标 🎲 可随机生成评价文案

菜品评价（选填）：

  1. 点击「添加菜品」按钮
  2. 输入菜名（必填）、评分（0-10 分）、一句话锐评（选填）
  3. 可上传菜品照片
  4. 可添加多道菜品

快速添加：页面会显示「这家店大家还评过」的菜品标签，点击可直接添加。

提交：点击「鉴定完毕」提交完整评价，或点击「仅鉴定餐厅」跳过菜品评价直接提交。

5. 广场

广场是查看所有人公开评价的社区页面，点击底部 Tab「广场」进入。

浏览评价

  • 上下滑动浏览瀑布流评价卡片
  • 每张卡片显示：档位色背景、评价内容、餐厅名、用户昵称/头像/称号、有品数
  • 下拉刷新，滚动到底自动加载更多

搜索与筛选

  • 搜索：顶部输入关键词，回车搜索
  • 排序：点击「最新/最热」切换排序
  • 筛选：点击「城市/等级/分类」按钮筛选

有品投票

点击卡片右下角的「有品」按钮表示你觉得这条评价中肯、有参考价值。再次点击取消投票。

6. 餐厅详情

点击任何餐厅卡片进入详情页。

顶栏操作

  • ← 返回：返回上一页
  • 书签图标：标记想去（再点取消）
  • 分享图标：生成分享海报
  • 更多图标 ⌄：反馈问题

店铺评价 Tab

  • 查看所有用户对该店的评价
  • 支持按「最新/最热」排序，按档位筛选
  • 每条评价可进行「有品/野榜」投票

菜品评价 Tab

  • 按菜品分组展示评价
  • 每道菜显示封面图、菜名、平均分、评价数
  • 点击菜品跳转菜品详情页

7. 菜品详情

在餐厅详情页的菜品评价 Tab 中，点击某道菜进入菜品详情页。

  • 显示菜品封面图、菜名、所属餐厅、平均分、评价数
  • 可切换排序（最新/最热/高分）
  • 每条评价可进行「有品/野榜」投票

8. 个人中心

点击底部 Tab「我的」进入。

个人资料

  • 显示头像、昵称、食鉴号、个性签名、食鉴数、有品数
  • 点击「编辑资料」可修改头像、昵称、签名等信息

评价记录

  • 查看所有已提交的食鉴记录
  • 点击餐厅名称可跳转餐厅详情
  • 点击每条记录的「···」菜单：
    • 设为私密/公开：控制评价是否对他人可见
    • 删除评价：删除后不可恢复

我的标记

  • 未鉴定：已标记想去但还未去的门店
  • 已鉴定：已完成食鉴的门店

我的伯乐

查看自己作为「首次鉴定者」的门店记录。伯乐记录由系统自动生成。

我的称号

  • 查看已获得的称号
  • 点击「佩戴」展示在广场、热评等处
  • 称号稀有度：普通 → 稀有 → 史诗 → 传说

应用设置

  • 设置密码：为手机号注册的账号设置密码
  • 隐私政策：查看隐私政策
  • 用户协议：查看用户协议
  • 退出登录：退出当前账号
  • 注销账号：永久删除账号及所有数据（不可恢复）

9. 分享功能

在餐厅详情页点击分享按钮，生成包含以下信息的分享海报：

  • 餐厅封面图 + 等级印章
  • 店名、分类、地址
  • 热评内容
  • 二维码（扫描可查看该餐厅）

选择「分享海报」发送给好友，或「保存到相册」留存。

10. 通知

  • 点击个人中心右上角铃铛图标进入通知页
  • 未读通知会显示红点角标
  • 点击通知查看详情
  • 未读通知自动标记为已读

11. 常见问题

Q：搜索不到我想评价的餐厅怎么办？

在搜索页面点击「手动补充」，填写餐厅名称、分类，并在地图上选点即可。评价提交后该餐厅会被收录。

Q：评价可以修改吗？

可以。进入餐厅详情页，点击底部「更新评价」按钮，重新走定档流程即可覆盖之前的评价。

Q：评价可以删除吗？

可以。进入「我的」→「评价记录」，点击对应记录的「···」菜单，选择「删除评价」。删除后不可恢复。

Q：什么是「有品」和「野榜」？

这是对评价内容的投票机制。有品：你觉得这条评价中肯、有参考价值。野榜：你觉得这条评价离谱、参考价值低。

Q：什么是伯乐？

伯乐是系统自动认定的「首家有效鉴定者」——第一个对某家餐厅完成食鉴的用户，其昵称会展示在餐厅详情页。

Q：如何让我的评价不被别人看到？

进入「我的」→「评价记录」，点击对应记录的「···」菜单，选择「设为私密」。私密评价仅自己可见。

Q：注销账号后数据能恢复吗？

不能。注销账号将永久删除所有个人数据，包括资料、评价、标记、投票、关注等，操作不可撤销。

Q：支持哪些登录方式？

支持手机号+密码登录和短信验证码登录。
